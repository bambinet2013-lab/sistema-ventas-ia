# 🏪 SISTEMA DE VENTAS CON IA AVANZADA

## 📋 DESCRIPCIÓN
Sistema de ventas con inteligencia artificial integrada que aprende de cada producto y se adapta al contexto venezolano.

## 🚀 CARACTERÍSTICAS IMPLEMENTADAS

### 🔴 PRIORIDAD 1: Búsqueda por similitud (LIKE)
- ✅ Búsqueda flexible de productos
- ✅ Detecta "PASTILLAS" aunque haya aprendido "PASTILLA"
- ✅ Resultados ordenados por relevancia

### 🔴 PRIORIDAD 2: Ignorar tildes y mayúsculas
- ✅ Normalización automática de texto
- ✅ "CAFÉ" = "CAFE", "PIÑA" = "PINA"
- ✅ Insensible a mayúsculas/minúsculas

### 🟡 PRIORIDAD 3: Análisis de marcas
- ✅ 48+ marcas venezolanas en base de datos
- ✅ Detección automática por marca
- ✅ Categorización inteligente

### 🟡 PRIORIDAD 4: Ponderación por frecuencia
- ✅ Pesos dinámicos por uso
- ✅ Palabras más usadas tienen prioridad
- ✅ Máximo peso: 5.0

### 🟢 PRIORIDAD 5: Relaciones entre palabras
- ✅ Familias de palabras (LÁCTEOS, FRENOS, etc.)
- ✅ Sinónimos agrupados
- ✅ 12 familias predefinidas

### 🟢 PRIORIDAD 6: Contexto de código de barras
- ✅ Detección de país por prefijo GS1
- ✅ 50+ países soportados
- ✅ Productos importados → Impuesto General
